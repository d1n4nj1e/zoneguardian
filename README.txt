# MineGuardian

## App Description
A simple mining safety monitoring app (MVP).

## MVP Scope
- User authentication
- Report safety issues
- View safety dashboard

## User Roles
- Worker
- Supervisor

## Core User Flow
Worker → Report issue → Supervisor reviews → Status updated

## Built with
Lovable AI
